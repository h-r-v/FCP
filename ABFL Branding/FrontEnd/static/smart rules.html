RULES TO USE SMART FCP

Preview Options Button

Use this button to see all available features for the document you have pasted the link for.

Execute FlowChart

Use this button to execute the flowchart you have designed.

SWIMLANE:

- Put GSUIT related links in these blocks. - Google Documents and Google Sheets are supported by this application. - Make sure to make your google document/sheet publicly accessible before pasting in the link.

RECTANGLE:

- Put queries related to the document that is attached to this box. - Make sure the queries you are passing here is supported by the application.
